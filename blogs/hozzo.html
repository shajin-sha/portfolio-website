About Hozzo

Hozzo is India's No. 1 car wash app developed using Flutter, offering a seamless and convenient car washing experience.


Key Features

1. Flutter MVVM architecture using provider for efficient state management.

2. REST API integration for smooth data communication. Dry and readable code maintained using Freezed.

3. Developed within just 2 days, showcasing rapid development capabilities.

4. Currently serving over 10k users on Google Play Store and 5k users on the App Store.

5. Online payment options via RazorPay for hassle-free transactions.

6. Address and vehicle management, making it easy for users to add and manage their information.

7. Booking and cancellation of car wash services with a user-friendly interface.

8. Integration with Telegram bot for instant communication and updates.

9. Address-wise slot allocation for efficient scheduling.

10. Vehicle type pricing, offering different prices for SUVs, hatchbacks, and sedans.

11. Enhanced user experience (UX) for a seamless app interaction.


Technologies Used

1. Flutter framework for iOS and Android app development.

2. Provider package for MVVM state management.

3. REST API integration for data communication.

4. Freezed for maintaining clean datasets.

5. RazorPay for online payment processing.


Development Process

1. Idea and product planning using Notion.

2. Designing the app's prototype with Figma.

3. Code development with GitHub for version control.

4. Task management and workflow planning with Notion.
